"A vehicle garage entry/exit detection system based on mobile sensor data. It integrates numerical analysis methods (SVD denoising, QR feature selection) with machine learning to accurately classify vehicle status."

"基于手机多传感器数据的车辆地库进出检测系统。项目融合了 SVD 降噪、QR 分解特征筛选等数值计算方法与机器学习模型，实现了对车辆进库、出库及行驶状态的精准识别。"
